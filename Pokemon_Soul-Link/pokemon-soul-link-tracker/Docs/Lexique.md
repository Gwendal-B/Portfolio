# 📘 Lexique Officiel – PokéChallenge Tracker

Ce document définit la terminologie officielle utilisée dans le projet **PokéChallenge Tracker**. Il garantit la cohérence entre les pages informatives, le Pokédex et le tracker interactif.

---

## 📊 1. Terminologie du Pokédex

Ces termes désignent les caractéristiques d’une espèce de Pokémon.

| Terme | Définition |
|-------|------------|
| Nom | Nom de l’espèce du Pokémon. |
| Numéro du Pokédex | Identifiant officiel du Pokémon dans le Pokédex. |
| Génération | Génération d’apparition du Pokémon. |
| Type / Types | Catégorie élémentaire du Pokémon (Feu, Eau, Plante, etc.). |
| Statistiques de base | Valeurs fixes propres à chaque espèce. |
| PV | Points de Vie (HP). |
| Attaque | Puissance offensive physique. |
| Défense | Résistance aux attaques physiques. |
| Vitesse | Rapidité du Pokémon en combat. |
| Statistique spéciale | Statistique unique en Génération 1. |
| Attaque Spéciale | Introduite à partir de la Génération 2. |
| Défense Spéciale | Introduite à partir de la Génération 2. |
| Taux de capture | Probabilité de capture d’un Pokémon. |
| Évolutions | Formes évolutives du Pokémon. |
| Sprite | Image officielle du Pokémon. |

---

## 🎮 2. Terminologie du Tracker

Ces termes sont utilisés pour suivre la progression d’une run.

| Terme | Définition |
|-------|------------|
| Run | Partie de challenge en cours. |
| Nom de la run | Nom attribué à une aventure. |
| Mode de jeu | Type de challenge (Nuzlocke, Soul Link, Randomizer). |
| Génération | Génération utilisée dans la run. |
| Jeu | Version Pokémon jouée. |
| Joueur | Participant à la run. |
| Pokémon capturé | Pokémon obtenu durant la run. |
| Espèce | Nom du Pokémon capturé. |
| Surnom | Nom personnalisé donné au Pokémon. |
| Zone de capture | Lieu où le Pokémon a été capturé. |
| Statut | État du Pokémon durant la run. |
| En équipe | Pokémon actuellement utilisé. |
| Dans la boîte | Pokémon stocké dans le PC. |
| Mort | Pokémon mis hors combat de façon permanente. |
| Inutilisable | Pokémon ne pouvant plus être utilisé. |
| Lien Soul Link | Association entre deux Pokémon de joueurs différents. |

---

## 🔗 3. Terminologie des Règles

Ces termes sont utilisés dans les guides et les paramètres de run.

| Terme | Définition |
|-------|------------|
| Règles de base | Ensemble des règles principales d’un challenge. |
| Variantes | Modifications optionnelles des règles. |
| Règles personnalisées | Paramètres définis par l’utilisateur. |
| Nuzlocke | Mode de jeu avec mort permanente et capture limitée. |
| Soul Link | Mode coopératif reliant les Pokémon de deux joueurs. |
| Randomizer | Mode modifiant aléatoirement les éléments du jeu. |
| Dupes Clause | Empêche la capture de Pokémon déjà obtenus. |
| Shiny Clause | Autorise la capture des Pokémon chromatiques. |
| Level Cap | Limite de niveau imposée. |
| Hardcore Nuzlocke | Version plus exigeante du Nuzlocke. |
